# App
A mobile App made in Python and Kivy.


## Database
Raffa deve creare il database, per ora ci sono solo 2 tabelle:

La tabella utente:

|idUtente| nome | cognome | username | password |
|--------|------|---------|----------|----------|
|        |      |         |          |          |

E la tabella categorie:

|idCategoria| idUtente* | categoria |
|-----------|-----------|-----------|
|           |           |           |



# Utilizzo
Per utilizzare l'applicazione copiare le cartelle config, kv, schermate e il file main, eseguendo tutto dal main. Prima di fare questo però bisogna aver creato il database sul proprio Xampp.
